# Dual-Guided-Learning
Code accompanying the paper "A Dual Perspective on Decision-Focused Learning: Scalable Training via Dual-Guided Surrogates" by Paula Rodriguez-Diaz, Kirk Bansak, and Elisabeth Paulson

## Set up environment

To run the code, first you have to set up a conda environment. Once you have [Anaconda installed](https://docs.conda.io/projects/conda/en/latest/user-guide/install/index.html), run the following command:
```
conda env create --name dglenv --file=environment.yml
```
Once the environment has been created, load it using the command
```
conda activate dglenv
```
